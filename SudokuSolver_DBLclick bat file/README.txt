Sudoku Solver
by John Wiesenfeld

To run: double click SudokuSolver.bat

Enter a number key and press enter to make selections from the menu.

e.g., press "1" and then "enter" to load a puzzle from file.
You will then be prompted for the name of the file. Type the name of the file,
e.g., "input.txt" and press "enter". 


Format of input file:
must be of .txt file, containing only integers, separated by spaces.
For example:

0 1 0 0 0 2 0 0 0
0 0 0 9 8 0 0 7 0
8 0 4 6 7 0 0 9 0
0 7 1 0 6 0 9 0 0
2 0 0 0 9 0 0 0 4
0 0 9 0 1 0 5 8 0
0 2 0 0 4 6 8 0 9
0 8 0 0 2 9 0 0 0
0 0 0 8 0 0 0 5 0

***Store input file in same directory as SudokuSolver.bat and SudokuSolver.jar***
